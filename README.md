<h1 align="center">Among Us - Java Text-based Game</h1>

This is a text-based game inspired by the popular multiplayer game Among Us. The game consists of two types of players: BlueAstronauts (Crewmates) and RedAstronauts (Impostors). The RedAstronauts are impostors who try to sabotage and freeze the BlueAstronauts, while the BlueAstronauts aim to complete tasks and find the impostor among them.

<h2>Classes</h2>

- Player.java: This is the base class with common attributes and methods shared by all players.
- BlueAstronaut.java (Crewmate.java): This class extends Player.java and represents a BlueAstronaut (Crewmate). BlueAstronauts can complete tasks and call emergency meetings.
- RedAstronaut.java (Impostor.java): This class extends Player.java and represents a RedAstronaut (Impostor). RedAstronauts can sabotage and freeze the BlueAstronauts.
- Gameplay.java: This class is the driver class and it will run the simulation.

<h2>Installation and Execution</h2>

Ensure you have Java installed on your machine.
* Compile the Java files:
`javac Player.java BlueAstronaut.java RedAstronaut.java Gameplay.java`
* Run the Gameplay file to start the game:
`java Gameplay`

Basic tests to understand the working of Amidst Us are provided in Gameplay.java. Feel free to modify and add your own tests for comprehensive coverage.
